# Управление версиями МАФ-Клуб

## Текущая версия: v1.0.6

## Как обновить версию:

1. Увеличить номер версии на 1 (например, v1.0.6 → v1.0.7)
2. Найти и заменить в файлах:
   ```bash
   # Заменить v1.0.6 на v1.0.7 во всех HTML файлах
   sed -i 's/v1\.0\.6/v1.0.7/g' *.html
   ```
3. Закоммитить с указанием новой версии:
   ```bash
   git add *.html VERSION.md
   git commit -m "Update to v1.0.7 - [описание изменений]"
   git push
   ```

## Где находится версия:

- `index.html` - внизу справа
- `rating.html` - внизу справа
- `player.html` - внизу справа
- `login.html` - внизу справа
- `game-input.html` - внизу справа

Версия отображается в правом нижнем углу полупрозрачным текстом.
